# Rules

Bash command execute at the current working directory no need to cd.

**ACP spelling:** The protocol name is **ACP** (Agent Client Protocol). Do not write **apc** in prose, comments, UI copy, or docs—use **ACP**. The product surface is **ACP UI** (not “APC UI”). In code, the `AcpUi` / `acpUi` prefix is the conventional camelCase/PascalCase form of that name.
